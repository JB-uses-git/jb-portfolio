# Jalad Bhairao - Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, and vanilla JavaScript.

## Features

- 🎨 Modern UI/UX with gradient backgrounds and smooth animations
- 📱 Fully responsive design for all devices
- ⚡ Smooth scrolling and section navigation
- 🎭 Interactive elements with hover effects
- 🌈 Dynamic typing effect in hero section
- 📊 Organized sections for About, Skills, Projects, Education, and Contact
- 🎯 Optimized performance with debounced scroll handlers
- ♿ Accessible navigation and semantic HTML

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, Grid, Flexbox, animations
- **JavaScript (ES6+)** - Intersection Observer API, smooth scrolling, DOM manipulation
- **Google Fonts** - Inter & Poppins

## Project Structure

## Sections

1. **Hero Section** - Introduction with animated typing effect

## Key Features Explained

### Smooth Scrolling
Navigation links smoothly scroll to their respective sections with offset for the fixed navbar.

### Typing Effect
The hero subtitle cycles through multiple titles with a realistic typing animation.

### Scroll Animations
Elements fade in and slide up as they enter the viewport using Intersection Observer API.

### Responsive Navigation
Mobile-friendly hamburger menu that transforms navigation on smaller screens.

### Parallax Effect
Subtle parallax scrolling on gradient orbs in the hero section.

### Project Card Tilt
3D tilt effect on project cards that responds to mouse movement.

## Customization

### Update Colors
Edit CSS custom properties in `styles.css`:

```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    /* ... more colors */
}
```

### Update Content
Modify the HTML content in `index.html` to reflect your information:
- Personal details in the hero section
- Projects in the projects section
- Skills in the skills section
- Contact information

### Update Links
Replace placeholder links with your actual profiles:
- GitHub: Update `href` in social links
- LinkedIn: Update `href` in social links
- Email: Update `mailto:` link
